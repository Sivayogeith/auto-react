# These Scripts setup entire MERN app for you
Run the command
 ```bash
bash ./MERN.sh
``` 
After Completion :

1. Run MongoDB 
2. Go to Server and run
 ```node
node index.js
```  
3. Go to Client and run 
```
npm run dev
```
